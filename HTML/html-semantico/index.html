Parrafo 1

El HTML semantico es una forma de escribir codigo HTML utilizando etiquetas que describen el contenido que contienen, esto hace que el codigo sea mas legible y facil de entender para los desarrolladores y para los motores de busqueda.

Parrafo 2

El HTML semantico tambien ayuda a mejorar la accesibilidad de las paginas web, ya que los lectores de pantalla pueden interpretar mejor el contenido y presentarlo de una manera mas comprensible para los usuarios con discapacidades visuales.

Parrafo 3

En resumen, el HTML semantico es una buena practica que mejora la legibilidad, la accesibilidad y el SEO de las paginas web, por lo que es recomendable utilizarlo en nuestros proyectos web.
